6.6. Diffutils-3.8

Le paquet Diffutils contient les programmes montrant les différences entre fichiers ou répertoires.

Temps de construction approximatif: 0.2 SBU
Espace disque requis: 28 Mo

6.6.1. Installation de Diffutils

Préparez la compilation de Diffutils : 

./configure --prefix=/usr --host=$LFS_TGT

Compilez le paquet : 

make

Installez ce paquet : 

make DESTDIR=$LFS install

Les détails de ce paquet se trouvent sur Section 8.55.2, « Contenu de Diffutils. »
