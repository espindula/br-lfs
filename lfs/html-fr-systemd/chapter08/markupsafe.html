8.69. MarkupSafe-2.0.1

MarkupSafe est un module Python qui implémente les chaînes sures pour le balisage XML/HTML/XHTML.

Temps de construction approximatif: moins de 0.1 SBU
Espace disque requis: 516 Ko

8.69.1. Installation de MarkupSafe

Compilez MarkupSafe avec la commande suivante : 

python3 setup.py build

Ce paquet n'a pas de suite de tests.

Installez le paquet : 

python3 setup.py install --optimize=1

8.69.2. Contenu de MarkupSafe

Répertoire installé: /usr/lib/python3.9/site-packages/MarkupSafe-2.0.1-py3.9.egg
