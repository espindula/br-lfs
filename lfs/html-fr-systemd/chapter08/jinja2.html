8.70. Jinja2-3.0.1

Jinja2 est un module Python qui implémente un langage de modèle en python.

Temps de construction approximatif: moins de 0.1 SBU
Espace disque requis: 3.7 Mo

8.70.1. Installation de Jinja2

Installez le paquet : 

python3 setup.py install --optimize=1

8.70.2. Contenu de Jinja2

Répertoires installés: /usr/lib/python3.9/site-packages/Jinja2-3.0.1-py3.9.egg
