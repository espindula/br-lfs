Ark-21.08.0

Introduction à Ark

Le paquet Ark est un outil d'archivage de KF5. Il s'agit d'une interface graphique à tar et des outils similaires.

This package still needs to be checked.

Dépendances de Ark

Requises

KDE Frameworks-5.85.0 et libarchive-3.5.2

Recommandées

cpio-2.13, p7zip-17.04, UnRar-6.0.7, UnZip-6.0 et Zip-3.0

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/ark5

Installation de Ark

Installez Ark en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
