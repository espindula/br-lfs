Dépendances de Polkit-Qt

Requises

CMake-3.21.2, Polkit-0.119 et Qt-5.15.2

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/polkit-qt

Installation de Polkit-Qt

Installez Polkit-Qt en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DCMAKE_BUILD_TYPE=Release : Ce paramètre est utilisé pour appliquer un plus haut niveau d'optimisation du compilateur.

Contenu

Programmes installés: aucun
Bibliothèques installées: libpolkit-qt5-agent-1.so, libpolkit-qt5-core-1.so et libpolkit-qt5-gui-1.so
Répertoires installés: /usr/include/polkit-qt5-1 et /usr/lib/cmake/PolkitQt5-1
