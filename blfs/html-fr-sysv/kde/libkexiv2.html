Introduction à libkexiv2

Libkexiv2 est une enveloppe KDÊ autour de la bibliothèque Exiv2 pour manipuler les métadonnées des images.

Installation de libkexiv2

Installez libkexiv2 en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
