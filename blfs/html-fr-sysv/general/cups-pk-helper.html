cups-pk-helper-0.2.6

Introduction à cups-pk-helper

Le paquet cups-pk-helper contient une aide à PolicyKit utilisée pour configurer Cups avec des privilèges fins.

Installation de cups-pk-helper

Tout d'abord, corrigez certaines syntaxe obsolètes qui empêcheront de construire cups-pk-helper avec les futures versions de CUPS : 

patch -Np1 -i ../cups-pk-helper-0.2.6-consolidated_fixes-1.patch

Installez cups-pk-helper en lançant les commandes suivantes : 

./configure --prefix=/usr --sysconfdir=/etc &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
