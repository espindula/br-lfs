Installation de Babl

Installez Babl en lançant les commandes suivantes : 

mkdir bld &&
cd    bld &&

meson --prefix=/usr --buildtype=release .. &&
ninja

Pour tester les résultats lancez : ninja test.

Maintenant, en tant qu'utilisateur root : 

ninja install &&

install -v -m755 -d                         /usr/share/gtk-doc/html/babl/graphics &&
install -v -m644 docs/*.{css,html}          /usr/share/gtk-doc/html/babl          &&
install -v -m644 docs/graphics/*.{html,svg} /usr/share/gtk-doc/html/babl/graphics

Explication des commandes

install -v -m755 -d /usr/share/gtk-doc/html/babl/graphics : Ceci ainsi que les commandes qui en résultent installent la documentation html de la bibliothèque dans /usr/share/gtk-doc/html où d'autres paquets gtk placent leur documentation dédiée aux programmeurs.
