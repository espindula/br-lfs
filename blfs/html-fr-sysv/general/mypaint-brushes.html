Introduction à mypaint-brushes

Le paquet mypaint-brushes contient les pinceaux utilisés par les paquets qui utilisent libmypaint.

Installation de mypaint-brushes

Installez mypaint-brushes en lançant les commandes suivantes : 

patch -Np1 -i ../mypaint-brushes-1.3.0-automake_1.16-1.patch &&
./autogen.sh                                                 &&
./configure --prefix=/usr                                    &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
