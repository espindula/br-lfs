Introduction à LXMenu Data

Le paquet LXMenu Data fournit les fichiers requis pour construire les menus compatibles avec la spécification freedesktop.org menu pour LXDE.

This package still needs to be checked.

Installation de LXMenu Data

Installez LXMenu Data en lançant les commandes suivantes : 

./configure --prefix=/usr --sysconfdir=/etc &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
