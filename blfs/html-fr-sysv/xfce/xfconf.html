Installation de Xfconf

Installez Xfconf en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contenu

Descriptions courtes

xfconf-query

est un outil en ligne de commande pour voir ou modifier un paramètre stocké dans xfconf

libxfconf-0.so

contient les fonctions de base de la configuration de Xfce
