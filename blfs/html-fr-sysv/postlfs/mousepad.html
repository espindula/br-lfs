Introduction à Mousepad

Mousepad est un éditeur de textes simple utilisant GTK+ 3 pour l'environnement de bureau Xfce.

Installation de Mousepad

Installez Mousepad en lançant les commandes suivantes : 

./configure --prefix=/usr --enable-keyfile-settings &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--enable-keyfile-settings : Utilisez le moteur keyfile de GSettings plutôt que DConf-0.40.0 par défaut.
