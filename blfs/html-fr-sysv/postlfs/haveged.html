Haveged-1.9.14

Introduction à Haveged

Le paquet Haveged contient un démon qui génère un flux imprévisible de nombres aléatoires et alimente le périphérique /dev/random.

Installation de Haveged

Installez Haveged en lançant les commandes suivantes : 

./configure --prefix=/usr --disable-static &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install &&
mkdir -pv    /usr/share/doc/haveged-1.9.14 &&
cp -v README /usr/share/doc/haveged-1.9.14

Configuration de haveged

Script de démarrage

Si vous voulez que le démon Haveged démarre automatiquement quand le système démarre, installez le script d'init /etc/rc.d/init.d/haveged inclus dans le paquet blfs-bootscripts-20210810 (en tant qu'utilisateur root) : 

make install-haveged

Descriptions courtes

haveged

est un démon qui génère un flux imprévisible de nombres aléatoires récoltés des effets indirects d'évènements matériels basés sur les états cachés du processeur (caches, prédicteurs de branchement, des tables de traduction de la mémoire, etc)
