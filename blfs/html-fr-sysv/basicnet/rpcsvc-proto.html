rpcsvc-proto-1.4.2

Introduction à rpcsvc-proto

Le paquet rpcsvc-proto contient les fichiers et les en-têtes rcpsvc, précédemment inclus dans glibc et qui ne sont pas inclus dans le paquet libtirpc-1.3.2 de remplacement, ainsi que le programme rpcgen.

This package still needs to be checked.

Installation de rpcsvc-proto

Installez rpcsvc-proto en lançant les commandes suivantes : 

./configure --sysconfdir=/etc &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
