gdk-pixbuf-xlib-2.40.2

Introduction gdk-pixbuf-xlib

Le paquet gdk-pixbuf-xlib fournit une interface Xlib obsolète à gdk-pixbuf, qui est requis pour certaines applications qui n'ont pas encore été portées pour utiliser les nouvelles interfaces.

This package still needs to be checked.

Installation de gdk-pixbuf-xlib

Installez gdk-pixbuf-xlib en lanant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr .. &&
ninja

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

ninja install

Explication des commandes

-Dgtk_doc=true : utilisez cette option si GTK-Doc-1.33.2 est installé et que vous souhaitez reconstruire et installer la documentation de l'API.
