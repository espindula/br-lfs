Installation de GTK Engines

Installez GTK Engines en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--enable-lua --with-system-lua: Utilisez ces paramètres si vous avez installé Lua et voulez construire avec le moteur de thème Lua.
