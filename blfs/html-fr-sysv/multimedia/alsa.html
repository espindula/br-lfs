ALSA-1.2.5

Le noyau Linux fournit maintenant par défaut le support d'ALSA. Cependant, des applications ont besoin de s'interfacer avec cette fonctionnalité. Les six sections suivantes du livre traitent des composants séparés d'ALSA : les bibliothèques, les greffons, les outils, les micro-codes et les bibliothèques de compatibilité OSS.
