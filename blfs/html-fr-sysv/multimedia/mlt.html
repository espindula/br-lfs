MLT-7.0.1

Introduction à MLT

Le paquet MLT est le Media Lovin Toolkit. C'est un outil libre multimédia, conçu et développé pour la diffusion télévisuelle. Il fournit un ensemble d'outils pour diffuseurs, éditeurs vidéo, lecteurs, transcodeurs, web streamers et d'autres types d'applications.

Installation de MLT

Installez MLT en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      -Wno-dev .. &&

make

Ce paquet n'a pas de suite de tests. Cependant, un fichier .mp4 de test peut être joué dans un environnement graphique avec ./src/melt/melt <filename>.mp4.

Note

Cette application utilise des fonctionnalités graphiques avancées. Dans certains cas, le micro-code de votre adaptateur graphique peut être demandé. Voir la section intitulée « Firmware pour les cartes vidéo » pour plus d'information.

Maintenant, en tant qu'utilisateur root : 

make install
