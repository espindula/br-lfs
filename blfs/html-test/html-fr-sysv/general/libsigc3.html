libsigc++-3.0.7

Introduction libsigc++3

Le paquet libsigc++3 implémente un système d'appel retour typé pour le C++ standard.

This package is known to build and work properly using an LFS-11.0 platform.

Installation de libsigc++3

Installez libsigc++3 en lanant les commandes suivantes : 

mkdir bld &&
cd    bld &&

meson --prefix=/usr --buildtype=release .. &&
ninja

Pour tester les rsultats lancez : ninja test.

Maintenant, en tant qu'utilisateur root : 

ninja install

Explication des commandes

-Dbuild_documentation=true : utilisez ce paramètre si Doxygen-1.9.2 est installé et que vous vouez construire et installer la documentation de l'API.
