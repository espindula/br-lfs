libuv-1.42.0

Introduction à libuv

Le paquet libuv est une bibliothèque de support multi-plateforme qui se concentre sur les entrées-sorties asynchrones.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

Installation de libuv

Installez libuv en lançant les commandes suivantes :

Attention

La commande sh autogen.sh ci-dessous échoue si la variable d'environnement ACLOCAL est présente telle que demandée dans Xorg-7. Si vous l'utilisez, vous devez désactiver la variable ACLCAL pour ce paquet et la réinitialiser pour les autres paquets. 

sh autogen.sh                              &&
./configure --prefix=/usr --disable-static &&
make

Si vous voulez lancer les tests, lancez make check.

Maintenant, en tant qu'utilisateur root : 

make install
