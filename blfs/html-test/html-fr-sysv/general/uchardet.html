Uchardet-0.0.7

Introduction Uchardet

Le paquet Uchardet contient une bibliothèque de détection de l'encodage qui prend une séquence d'octets dans un encodage inconnu et essaye de déterminer l'encodage du texte.

This package is known to build and work properly using an LFS-11.0 platform.

Dépendances de Uchardet

Requises

CMake-3.21.2

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/uchardet

Installation de Uchardet

Installez Uchardet en lanant les commandes suivantes : 

mkdir build &&
cd    build &&
     
cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DBUILD_STATIC=OFF          \
      -Wno-dev ..                 &&
make

Pour tester les rsultats lancez : make test.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DBUILD_STATIC=OFF : ce paramètre désactive la construction de la version statique de la bibliothèque.
