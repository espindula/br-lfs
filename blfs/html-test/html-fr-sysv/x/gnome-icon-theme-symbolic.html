Introduction à GNOME Icon Theme Symbolic

Le paquet GNOME Icon Theme Symbolic contient des icônes symboliques pour le thème d'icône GNOME par défaut.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

Dépendances de GNOME Icon Theme Symbolic

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/gnome-icon-theme-symbolic

Installation de GNOME Icon Theme Symbolic

Installez GNOME Icon Theme Symbolic en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
