gnome-icon-theme-3.12.0

Introduction à GNOME Icon Theme

Le paquet GNOME Icon Theme contient un assortiment d'icônes à taille fixe de différentes tailles et de différents thèmes.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Téléchargement (HTTP) : https://download.gnome.org/sources/gnome-icon-theme/3.12/gnome-icon-theme-3.12.0.tar.xz

  • Téléchargement (FTP) : ftp://ftp.acc.umu.se/pub/gnome/sources/gnome-icon-theme/3.12/gnome-icon-theme-3.12.0.tar.xz

  • Somme de contrôle MD5 du téléchargement : f14bed7f804e843189ffa7021141addd

  • Taille du téléchargement : 17 Mo

  • Estimation du temps de construction : 0.5 SBU

Dépendances de GNOME Icon Theme

Requises

GTK+-3.24.30 ou GTK+-2.24.33, hicolor-icon-theme-0.17 et icon-naming-utils-0.8.90

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/gnome-icon-theme

Installation de GNOME Icon Theme

Installez GNOME Icon Theme en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contenu

Répertoire installé: /usr/share/icons/gnome
