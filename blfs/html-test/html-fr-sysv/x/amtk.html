Amtk-5.2.0

Introduction Amtk

Le paquet Amtk contient un remplaçant à GTKUIManager basique basé sur GAction.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Tlchargement (HTTP) : https://download.gnome.org/sources/amtk/5.2/amtk-5.2.0.tar.xz

  • Tlchargement (FTP) : ftp://ftp.acc.umu.se/pub/gnome/sources/amtk/5.2/amtk-5.2.0.tar.xz

  • Somme de contrle MD5 du tlchargement : 8ea25cde8b7bdbf98d7d6c939dea1724

  • Taille du tlchargement : 376 Ko

  • Estimation de l'espace disque requis : 8.5 Mo

  • Estimation du temps de construction : 0.1 SBU

Dépendances de Amtk

Requises

GTK+-3.24.30

Facultatives

GTK-Doc-1.33.2 (pour la documentation) et Valgrind-3.17.0

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/amtk

Installation de Amtk

Installez Amtk en lanant les commandes suivantes : 

./configure --prefix=/usr &&
make

Pour tester les rsultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--enable-gtk-doc : Utilisez ce paramètre si GTK-Doc est installé et si vous souhaitez reconstruire et installer la documentation de l'API.
