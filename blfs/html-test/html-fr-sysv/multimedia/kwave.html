kwave-21.08.0

Introduction à KWave

Le paquet KWave contient une application d'édition audio basée sur KF5.

This package is known to build and work properly using an LFS-11.0 platform.

Installation de KWave

Installez KWave en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
