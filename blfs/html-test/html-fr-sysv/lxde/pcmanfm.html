PCManFM-1.3.2

Introduction à PCManFM

Le paquet PCManFM contient un gestionnaire de fichiers extrêmement rapide, léger, maintenant riche en fonctionnalité avec la navigation tabulée.

This package is known to build and work properly using an LFS-11.0 platform.

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/pcmanfm

Installation de PCManFM

Installez PCManFM en lançant les commandes suivantes : 

./configure --prefix=/usr     \
            --sysconfdir=/etc &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
