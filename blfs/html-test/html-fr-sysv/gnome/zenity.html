Zenity-3.32.0

Introduction à Zenity

Zenity est une réécriture de gdialog, le port GNOME de dialog qui permet d'afficher des boîtes de dialogue GTK+ depuis la ligne de commande et les scripts shell.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Téléchargement (FTP) : ftp://ftp.acc.umu.se/pub/gnome/sources/zenity/3.32/zenity-3.32.0.tar.xz

  • Somme de contrôle MD5 du téléchargement : ba2b2a13248773b4ec0fd323d95e6d5a

Installation de Zenity

Installez Zenity en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contents

Programmes installés: gdialog et zenity
Bibliothèques installées: Aucune
Répertoires installés: /usr/share/help/*/zenity et /usr/share/zenity

Descriptions courtes

gdialog

est un script enveloppe Perl qui peut être utilisé avec les scripts anciens

zenity

est un programme qui affichera des dialogues GTK+ et renverra l'entrée utilisateur
