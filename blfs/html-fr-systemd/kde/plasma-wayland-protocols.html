Plasma-wayland-protocols-1.3.0

Introduction à the Plasma-wayland-protocols

Ce paquet fournit un ensemble personnalisé de définitions pour KDE.

This package is known to build and work properly using an LFS-11.0 platform.

Dépendances de Plasma-wayland-protocols

Requises

extra-cmake-modules-5.85.0

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/plasma-wayland-protocols

Installation de Plasma-wayland-protocols

Installez Plasma-wayland-protocols en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr ..

Maintenant, en tant qu'utilisateur root : 

make install
