Introduction à libkcddb

Le paquet libkcddb contient une bibliothèque utilisée pour retrouver les métadonnées des CD audio depuis internet.

Installation de libkcddb

Installez libkcddb en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev ..                        &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
