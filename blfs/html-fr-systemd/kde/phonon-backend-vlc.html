Phonon-backend-vlc-0.11.3

Introduction à Phonon-backend-vlc

Ce paquet fournit un moteur Phonon qui utilise le framework média VLC.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Téléchargement (HTTP) : https://download.kde.org/stable/phonon/phonon-backend-vlc/0.11.3/phonon-backend-vlc-0.11.3.tar.xz

  • Taille du téléchargement : 68 Ko

Dépendances de Phonon-backend-vlc

Requises

phonon-4.11.1 et VLC-3.0.16

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/phonon-backend-vlc

Installation de Phonon-backend-vlc

Installez Phonon-backend-vlc en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      .. &&
make

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DCMAKE_BUILD_TYPE=Release : Ce paramètre est utilisé pour appliquer un plus haut niveau d'optimisation du compilateur.

Contenu

Programme installé: aucun
Bibliothèques installées: phonon_vlc.so
Répertoire installé: /usr/lib/qt5/plugins/phonon4qt5_backend
