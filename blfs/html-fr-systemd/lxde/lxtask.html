LXTask-0.1.10

Introduction à LXTask

Le paquet LXTask contient un gestionnaire de tâches léger indépendant du bureau.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Téléchargement (HTTP) : https://downloads.sourceforge.net/lxde/lxtask-0.1.10.tar.xz

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de LXTask

Requises

GTK+-2.24.33

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/lxtask

Installation de LXTask

Installez LXTask en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
