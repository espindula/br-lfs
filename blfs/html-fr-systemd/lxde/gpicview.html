Installation de GPicView

Installez GPicView en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install &&
sed -i 's/Utility;//' /usr/share/applications/gpicview.desktop

Si vous avez installé xdg-utils-1.1.3, en tant qu'utilisateur root, vous devriez lancer la commande xdg-icon-resource forceupdate --theme hicolor pour que les nouvelles icônes installées soient affichées dans les menus.

Explication des commandes

sed -i ... : Corrige les sous-menus pour GPicView.
