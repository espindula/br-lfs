Garcon-4.16.1

Introduction à Garcon

Le paquet Garcon contient une implémentation de menu conforme à freedesktop.org basée sur GLib et GIO.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

Dépendances de Garcon

Requises

libxfce4ui-4.16.0 et GTK+-3.24.30

Facultatives

gobject-introspection-1.68.0 et GTK-Doc-1.33.2

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/garcon

Installation de Garcon

Installez Garcon en lançant les commandes suivantes : 

./configure --prefix=/usr --sysconfdir=/etc &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
