xfce4-terminal-0.8.10

Introduction à Xfce4 Terminal

Xfce4 Terminal est un émulateur de terminal Gtk+3. Il sert à lancer des commandes ou des programmes avec l'avantage d'une fenêtre Xorg, vous pouvez glisser-déposer des fichiers dans le terminal xfce4 ou copier-coller du texte avec votre souris.

Installation de Xfce4 Terminal

Installez Xfce4 Terminal en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
