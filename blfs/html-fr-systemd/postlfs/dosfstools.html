Configuration du noyau

Activez les options suivantes dans la configuration du noyau et recompilez le noyau : 

File systems --->
  <DOS/FAT/EXFAT/NT Filesystems --->
    <*/M> MSDOS fs support             [CONFIG_MSDOS_FS]
    <*/M> VFAT (Windows-95) fs support [CONFIG_VFAT_FS]

Installation de dosfstools

Installez dosfstools en lanant les commandes suivantes : 

./configure --prefix=/usr            \
            --enable-compat-symlinks \
            --mandir=/usr/share/man  \
            --docdir=/usr/share/doc/dosfstools-4.2 &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--enable-compat-symlinks : Ce paramètre crée les liens symboliques dosfsck, dosfslabel, fsck.msdos, fsck.vfat, mkdosfs, mkfs.msdos et mkfs.vfat requis par certains programmes.
