Ed-1.17

Introduction à Ed

Ed est un éditeur orienté ligne de commande. Il est utilisé pour créer, afficher, modifier voire manipuler des fichiers textes, autant de façon interactive que via des scripts shell. Ed n'est pas utilisé par beaucoup de gens. On le décrit ici car le programme patch peut l'utiliser si vous rencontrez un fichier de correctif basé sur ed. Cela arrive rarement car on préfère de nos jours des correctifs basés sur diff.

Installation de Ed

Installez Ed en lançant les commandes suivantes : 

./configure --prefix=/usr --bindir=/bin &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install
