Sysprof-3.40.1

Introduction Sysprof

Le paquet sysprof contient un profileur statistique pour le système en entier, pour Linux.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Tlchargement (HTTP) : https://download.gnome.org/sources/sysprof/3.40/sysprof-3.40.1.tar.xz

  • Tlchargement (FTP) : ftp://ftp.acc.umu.se/pub/gnome/sources/sysprof/3.40/sysprof-3.40.1.tar.xz

  • Somme de contrle MD5 du tlchargement : 05d1399cb3ab923b03603b10acd8d40e

  • Taille du tlchargement : 460 Ko

  • Estimation de l'espace disque requis : 20 Mo

  • Estimation du temps de construction : 0.1 SBU (avec paralllisme = 4)

Dépendances de Sysprof

Requises

GTK+-3.24.30, JSON-GLib-1.6.6, libdazzle-3.40.0, et Polkit-0.119

Facultatives

libunwind

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/sysprof

Installation de Sysprof

Installez Sysprof en lanant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr .. &&
ninja

Pour tester les rsultats lancez : ninja test.

Maintenant, en tant qu'utilisateur root : 

ninja install

Contents

Programmes installs: sysprof et sysprof-cli
Bibliothques installes: libsysprof-4.so, libsysprof-capture-4.a, libsysprof-memory-4.so, libsysprof-speedtrack-4.so et libsysprof-ui-4.so
Rpertoires installs: /usr/include/sysprof-4 et /usr/share/help/*/sysprof

Descriptions courtes

sysprof

est une interface graphique pour sysprof

sysprof-cli

est une interface en ligne de commande pour sysprof

libsysprof-4.so

fournit les fonctions de l'API pour profiler les processus lancés sur un système

libsysprof-capture-4.a

fournit un crochet utilisé pour capturer les statistiques d'un processus

libsysprof-memory-4.so

fournit les fonctions de l'API pour capturer les statistiques mémoire

libsysprof-speedtrack-4.so

fournit les fonctions de l'API pour suivre la vitesse d'un processus en cours d'exécution

libsysprof-ui-4.so

fournit les fonctions de l'API pour l'interface sysprof
