Asciidoctor-2.0.16

Introduction Asciidoctor

Asciidoctor est un traitement de texte et chaîne d'outils de publication rapide et open source pour convertir du contenu AsciiDoc en HTML5, DocBook, PDF et d'autres formats.

Installation de Asciidoctor

Construisez la gem Ruby : 

gem build asciidoctor.gemspec

La suite de tests demande plusieurs gems Ruby en dehors de la portée de BLFS.

Maintenant, en tant qu'utilisateur root : 

gem install asciidoctor-2.0.16.gem &&
install -vm644 man/asciidoctor.1 /usr/share/man/man1
