Hd2u-1.0.4

Introduction à Hd2u

Le paquet hd2u contient un convertisseur de texte depuis et vers n'importe quel format.

This package still needs to be checked.

Dépendances de Hd2u

Requises

popt-1.18

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/hd2u

Installation de Hd2u

Installez hd2u en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
