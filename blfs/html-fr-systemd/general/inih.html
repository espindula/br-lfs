inih-53

Introduction inih

Ce paquet est un analyseur de fichier .INI simple écrit en C.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/benhoyt/inih/archive/r53/inih-r53.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 8b52bf308de2ad1eab94fad64c116610

  • Taille du téléchargement : 17 Ko

  • Estimation de l'espace disque requis : 0.6 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/inih

Installation de inih

Installez inih en lanant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr --buildtype=release .. &&
ninja

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

ninja install
