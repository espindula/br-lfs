Installation de Pnmixer

Installez Pnmixer en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr ..  &&
make

Ce paquet n'a pas de suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
