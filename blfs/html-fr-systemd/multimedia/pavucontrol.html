pavucontrol-5.0

Introduction à pavucontrol

PulseAudio Volume Control (pavucontrol) est un outil de contrôle de volume simple en GTK (« mixeur ») pour le serveur de son PulseAudio. Contrairement aux outils de mixage classique, celui-ci vous permet de contrôler aussi bien le volume des périphériques matériels que de chaque flux séparément.

This package is known to build and work properly using an LFS-11.0 platform.

Dépendances de pavucontrol

Requises

Gtkmm-3.24.5, libcanberra-0.30, libsigc++-2.10.7 et PulseAudio-15.0

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/pavucontrol

Installation de pavucontrol

Installez pavucontrol en lançant les commandes suivantes : 

./configure --prefix=/usr --docdir=/usr/share/doc/pavucontrol-5.0 &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
