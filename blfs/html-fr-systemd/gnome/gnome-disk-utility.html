Introduction à GNOME Disk Utility

Le paquet GNOME Disk Utility fournit les applications utilisées pour s'occuper des périphériques de stockage.

This package still needs to be checked.

Installation de GNOME Disk Utility

Installez GNOME Disk Utility en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr --buildtype=release .. &&
ninja

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

ninja install

gnome-disk-image-mounter

est utilisé pour mettre en place des images disques

gnome-disks

est utilisé pour inspecter, formater, partitionner et configurer des disques et des périphériques de blocs
