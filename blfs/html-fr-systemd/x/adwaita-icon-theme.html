Introduction à Adwaita Icon Theme

Le paquet Adwaita Icon Theme contient un thème d'icônes pour les applications Gtk+ 3.

This package still needs to be checked.

Dépendances de Adwaita Icon Theme

Facultatives

git-2.33.0, GTK+-2.24.33 ou GTK+-3.24.30 (si présent, librsvg-2.50.7 est aussi requis, et gtk-update-icon-cache et gtk-encode-symbolic-svg sont lancés après l'installation), Inkscape-1.1 et Icon Tools

Notes utilisateur : https://wiki.linuxfromscratch.org/blfs/wiki/adwaita-icon-theme

Installation de Adwaita Icon Theme

Installez Adwaita Icon Theme en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
