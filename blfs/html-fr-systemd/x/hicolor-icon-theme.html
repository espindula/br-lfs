Introduction à hicolor-icon-theme

Le paquet hicolor-icon-theme contient un thème de secours par défaut pour les implémentations de la spécification des thèmes d'icônes.

Installation de hicolor-icon-theme

Installez hicolor-icon-theme en lançant les commandes suivantes : 

./configure --prefix=/usr

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
