Installation de Graphene

Installez Graphene en lanant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr --buildtype=release .. &&
ninja

Pour tester les rsultats lancez : ninja test.

Maintenant, en tant qu'utilisateur root : 

ninja install

Explication des commandes

--buildtype=release : spécifie le type de construction convenant aux versions stables de ce paquet, comme la valeur par défaut produit des binaires non optimisés.

-Dgtk_doc=true . utilisez ce paramètre si vous avez installé GTK-Doc-1.33.2 et que vous souhaitez générer la documentation de l'API.
