Installation de neon

Installez neon en lançant les commandes suivantes : 

./configure --prefix=/usr    \
            --with-ssl       \
            --enable-shared  \
            --disable-static &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--with-ssl : cette option active la prise en charge de SSL en utilisant OpenSSL ou GnuTLS respectivement. Pour forcer l'utilisation de GnuTLS quand les deux sont présents, passez simplement --with-ssl=gnutls et --with-ca-bundle=/etc/pki/tls/certs/ca-bundle.crt au script configure.

--disable-static : Ce paramètre empêche l'installation des versions statiques des bibliothèques.
