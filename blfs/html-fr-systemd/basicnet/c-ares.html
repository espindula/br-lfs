c-ares-1.17.2

Introduction à c-ares

c-ares est une bibliothèque C pour les requêtes DNS asynchrones.

This package is known to build and work properly using an LFS-11.0 platform.

Informations sur le paquet

  • Estimation de l'espace disque requis : 11 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de c-ares

Requises

CMake-3.21.2

Installation de c-ares

Installez c-ares en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake  -DCMAKE_INSTALL_PREFIX=/usr .. &&
make

Ce paquet n'inclut pas de suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contents

Programmes installés: acountry, adig et ahost
Bibliothèques installées: libcares.so
Répertoires installés: None

Descriptions courtes

acountry

affiche les pays où une adresse IPv4 ou un hôte se trouve

adig

demande des informations à des serveurs DNS

ahost

affiche l'enregistrement A ou AAAA associé à un nom d'hôte ou une adresse IP

libcares.so

est une bibliothèque C pour les requêtes DNS asynchrones
