libmnl-1.0.4

Introduction à libmnl

La bibliothèque libmnl fournit une bibliothèque en espace utilisateur minimale orientée pour les développeurs Netlink. Il y a plein de tâches communes entre l'analyse, la validation, la construction d'en-têtes Netlink et de champs type-longueur-valeur qui sont répétitives et sur lesquelles il est facile de se tromper. Cette bibliothèque a pour but de fournir des fonctions d'aide pour vous permettre de réutiliser du code et éviter de réinventer la roue.

Installation de libmnl

Installez libmnl en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
